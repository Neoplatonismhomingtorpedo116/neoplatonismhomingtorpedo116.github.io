🛠 Troubleshooting

If you encounter issues during installation or runtime, consider these common solutions:

First, verify that your Windows system is fully updated. If the application refuses to open, check whether your user account has administrative permissions. Sometimes, security software might block unknown applications. You may need to click "Run anyway" if Windows displays a security alert during your first launch.

If the editor window appears blank, ensure you have the correct WebView2 runtime installed. You can download the latest runtime directly from the Microsoft website if the software installer did not complete this step automatically.
